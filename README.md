# Crystalline Max — Landing Page

Marketing landing page for **Crystalline Max Ltd**, a premium mobile car detailing and residential/commercial cleaning business operating across Manchester, Salford, Stockport, and Oxfordshire.

## Overview

A static, single-page website that drives all bookings through WhatsApp. No app, no accounts, no backend — just a clean marketing presence with a direct line to the business.

## Pages

| File | Description |
|---|---|
| `index.html` | Main landing page with hero slider, services, estimator, and CTAs |
| `privacy.html` | Privacy Policy (UK GDPR compliant) |
| `terms.html` | Terms of Service |

## Key Features

- **Hero Slider** — Auto-rotating full-section slides showcasing car detailing, home cleaning, and office cleaning
- **Cost Estimator** — Interactive price calculator with service selection, add-ons, and live total that pre-fills a WhatsApp message
- **WhatsApp-First** — Every CTA opens a WhatsApp chat with a pre-filled booking message
- **Fully Responsive** — Mobile hamburger nav, touch-friendly targets, breakpoints at 400px / 640px / 768px / 1024px
- **Zero Dependencies** — Pure HTML, CSS, and vanilla JS in a single file per page

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Montserrat + Inter)
- No build step, no frameworks, no npm
